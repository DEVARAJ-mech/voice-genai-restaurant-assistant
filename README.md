# Voice-Enabled GenAI Restaurant Assistant

This project implements a voice-driven Generative AI restaurant assistant that acts as a virtual receptionist.

## Features

- Voice-based customer interaction
- Table reservations
- Order handling
- Menu and allergen queries
- Offline LLM inference using Ollama

## Technology Stack

- Backend: FastAPI (Python)
- LLM: Ollama (Mistral)
- STT: Whisper (planned)
- TTS: Piper / Coqui (planned)
